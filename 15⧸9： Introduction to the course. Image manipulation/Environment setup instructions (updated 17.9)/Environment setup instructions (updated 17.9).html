The code for the first part of the course will only require a python 3 jupyter notebook with numpy, matplotlib and scikit-image.  To use widgets, we also install ipywidgets.

The recommended way to setup such an environment, that you should try in case you have issues with your installation, is the following.

  • Download the miniconda installer for the latest python 3 version: https://docs.conda.io/en/latest/miniconda.html. you can skip this step if you have anaconda already installed and working.
  • create a new conda environment; activate it. To do so, open a terminal and run the following. If conda is not found as a command, try explicitly opening a miniconda prompt or anaconda prompt.
    conda create -n cvdl
    conda activate cvdl
  • in the same terminal, in which the cvdl environment should now be active, run 
    conda install -c conda-forge jupyter scikit-learn scikit-image matplotlib ipywidgets ipykernel
  • launch the notebook: 
    jupyter lab

Many students successfully use a conda environment (or other virtual python environment) within Visual Studio Code. That is also OK and typically works fine.

After any lecture, the instructor is available for assistance with your setup.

In the second part of the course other libraries will be used for deep learning tasks (tensorflow and/or pytorch), and will be introduced in due time.
